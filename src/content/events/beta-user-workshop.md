## Beta User Workshop

We're hosting a virtual workshop for early adopters and beta users of the GP-STOR platform.

### What You'll Learn

- How to access GP-STOR through various methods (NRP mounts, Globus, Nextcloud)
- Best practices for data management and storage
- Performance optimization tips
- Q&A with the GP-STOR team

### Who Should Attend

This workshop is designed for researchers and system administrators who:
- Are interested in using distributed storage for their projects
- Need high-performance storage for AI/ML workloads
- Want to learn about modern storage infrastructure
