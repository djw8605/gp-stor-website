A comprehensive, hands-on training workshop designed specifically for GP-STOR beta users.

## What You'll Learn

This full-day workshop will cover:
- Advanced data management techniques
- Optimizing data transfer performance
- Integration with HPC workflows
- Monitoring and troubleshooting
- Best practices for distributed storage

**Time**: 10:00 AM - 4:00 PM CST  
**Location**: University of Nebraska, Lincoln - Holland Computing Center

- 10:00 AM - Welcome and Overview
- 10:30 AM - GP-STOR Architecture Deep Dive
- 12:00 PM - Lunch (provided)
- 1:00 PM - Hands-on Lab Session
- 3:00 PM - Q&A and Networking
- 4:00 PM - Closing

## Prerequisites

- Active GP-STOR beta user account
- Laptop with SSH client installed
- Basic familiarity with Linux command line
